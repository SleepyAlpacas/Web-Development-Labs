Lab03 Part 3

By Daniel Kong, Oct 21, 2022
School Web Development

The css implementation of the responsive site was much more tedious. I had to set up multiple classes before I could even get started with putting things on my website. The column system created with css is also less intuitive than the bootstrap version. Getting started with bootstrap was way easier in comparison to css, since all you needed to do was import the boostrap library. Once that was done, you could just use columns without having to create them manually, and they work better than anything I could make.

The challenge with the css version was wrapping my head around the grid system, but it was smooth sailing once I understood. Meanwhile I had basically no challenges at all with bootstrap. Bootstrap took less time for me and looked just as good with less effort. I definitely enjoyed working with bootstrap more.
