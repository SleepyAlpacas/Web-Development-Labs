How I built the website

I went on the Bulma website and used the "getting started" page to build my website. Installing bulma was as simple as including a link in the head of the html document and creating Bulma elements were equally easy, only needing to include a class to certain elements

Difficulties Encountered

Development of the website was mostly smooth, however implementing the button did take some reading outside of the Bulma documentation. The main difficulty was just formatting and understanding the code because of how many classes are used in Bulma.
